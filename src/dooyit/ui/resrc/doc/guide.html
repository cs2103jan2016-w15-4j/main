Add task

add <task name>
add <task name> by <datetime>
add <task name> from <datetime> to <datetime>

Delete task

delete <task id>
delete <task id> <task id> <task id>
delete <task id> - <task id>

Edit task

edit <task id> by <datetime>
edit <task id> <new task name> by <datetime>
edit <task id> from <datetime> to <datetime>
edit <task id> <new event name> from <datetime> to <datetime>
edittofloat <task id>
float <task id>
float <task id> <task id> <task id>
float <task id> - <task id>

Mark Task

mark <task id>
mark <task id> <task id> <task id>
mark <task id> - <task id>

Unmark Task

unmark <task id>
unmark <task id>-<task id>
unmark <task id> <task id>

Move task(s) to category

move <task id> <category name>
move <task id> <task id> <task id> <category name>
move <task id>-<task id> <category name>

Unmove (remove task from category)

unmove <task id>

Add category

addcat <category name>
addcat <category name> <color>
Note: The name of a category can only be a single word.
Valid colors: black, blue, cyan, grey, green, magenta, pink, red, yellow, white, orange

Delete category

deletecat <category name>

Edit category

editcat <current category name> <new category name>
editcat <category name> to <new color>
editcat <current category name> <new category name> to <new color>

undo

redo

showcat <category name>
