Add task or event

add <task name>
add <task name> by <datetime>

Add category

addcat <category name>
addcat <category name> <color>
Note: The name of a category can only be a single word.
Valid colors: black, blue, cyan, grey, green, magenta, pink, red, yellow, white

Clear

clear

Delete

delete <task id>
delete <task id> <task id>
delete <task id>-<task id>

Delete category

deletecat <category name>

Edit task or event

edit <task/event id> <new task name> by <datetime>
edit <task/event id> <new event name> from <datetime> to <datetime>
edit <task/event id> by <datetime>
edit <task/event id> from <datetime> to <datetime>

Edit category

editcat <old category name> <new category name> to <color>
editcat <old category name> <new category name>
editcat <category name> to <new color>
